# MySQL的语法解析器

SQL本身是一种程序语言, 语言最终会被编译成指令在程序中运行;

那么语言本身是可以被解析, 对应的解析器有哪些实现

1. MySQL源码的解析器
    
    由专门的公司维护 SQL Parser, https://www.sqlparser.com/
2. 开源的SQL解析器

   - 美团开源的SQL Advisor https://github.com/Meituan-Dianping/SQLAdvisor
